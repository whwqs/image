ArUco标记是可用于相机姿态估计的二进制正方形基准标记。它们的主要优点在于它们的检测是稳健，快速和简单的。

aruco模块包括这些类型的标记的检测和用于姿势估计和相机校准的工具。

此外，ChArUco功能将ArUco标记与传统棋盘结合在一起，可以方便多用的角落检测。该模块还包括检测ChArUco角落的功能，并将其用于姿态估计和相机校准。
